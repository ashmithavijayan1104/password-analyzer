# 🔐 Password Security Analyzer

A Streamlit-based ML + security tool that evaluates password strength and vulnerability.

## 🚀 Features
- Strength prediction (ML model)
- Dictionary attack detection
- Pattern-based risk detection
- Brute-force resistance
- Entropy calculation
- Smart recommendations

👉 https://password-analyzer-jhn6ykdgtlyuzswmburrpy.streamlit.app/

## 🛠 Tech Used
- Python
- Streamlit
- Scikit-learn
- Pandas
